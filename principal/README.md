# Caliandra Landpage

Página inicial do projeto Caliandra.

## Setup rápido

1. Criar e ativar ambiente virtual (Windows):
   ```bash
   python -m venv venv
   venv\\Scripts\\activate
   ```
2. Instalar dependências (ex.: Django):
   ```bash
   pip install django
   ```

## Desenvolvimento

- Código-fonte e assets ficam neste diretório `principal/`.
- Use o `.gitignore` para evitar subir arquivos locais e de build.

## Publicação

O repositório será criado no GitHub e conectado como `origin` para permitir `git push` na branch `main`.
